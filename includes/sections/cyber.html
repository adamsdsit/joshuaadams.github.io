Cybersecurity Fundamentals – Syllabus

Instructor: Dr. Joshua Adams

Term: Fall 2025

Course Description

This course introduces core cybersecurity principles, emphasizing hands-on, project-based learning using cloud operations. Topics include the CIA triad, threat modeling, risk management, cryptography, ethical hacking, and cyber law. Students will apply these concepts in real-world scenarios, utilizing cloud-based tools and platforms to design, implement, and defend secure systems.

Learning Objectives

  • Analyze threats and vulnerabilities in cloud-based environments and across network layers.
  • Apply cryptographic principles to secure communications and data within cloud infrastructures.
  • Utilize cloud-based security tools and frameworks (e.g., AWS Security Hub, Azure Security Center) in guided labs.
  • Design and assess risk mitigation strategies for cloud deployments, considering scalability and cost.
  • Implement and evaluate security best practices in a simulated cloud environment.
  • Conduct ethical hacking exercises in a controlled cloud setting to identify vulnerabilities.
  • Understand and apply cyber law and compliance regulations relevant to cloud operations.

Assessment & Grading

  • Cloud Security Labs & Practical Reports – 30%
  • Midterm Exam – 25%
  • Final Cloud Security Capstone Project – 30%
  • Participation & Discussions – 15%

Course Projects

This course includes a significant project component focused on real-world cloud security challenges. Students will work individually or in small groups to complete the following:

  • Cloud Infrastructure Security Design: Design a secure cloud infrastructure for a simulated organization, including network architecture, access controls, and data protection mechanisms.
  • Cloud Vulnerability Assessment and Penetration Testing: Perform a vulnerability assessment and penetration test on a deployed cloud application, documenting findings and recommending remediation strategies.
  • Incident Response in the Cloud: Develop an incident response plan for a cloud-based security breach, including steps for detection, containment, eradication, recovery, and post-incident analysis.
  • Secure Cloud Deployment and Automation: Automate the deployment of a secure cloud environment using Infrastructure as Code (IaC) and implement security best practices.
