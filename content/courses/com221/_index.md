### Course Description
Introduction to cybersecurity principles including risk management, network security, threat modeling, and defensive techniques.

### Learning Outcomes
- Understand core cybersecurity concepts
- Apply security tools in hands-on labs
- Analyze real-world security incidents
- Communicate risk and mitigation strategies

- Linux
- Cloud-based lab environments
